# Salary-and-Credit-Management-System
It is a basic web application that helps manage employee details, calculate salaries, and keep track of employee credits (such as salary advances). The project uses Google Sheets as the database and Google Apps Script for backend operations.

## What It Does

- Admin can log in using a username and password.
- Add, view, edit, and delete employee records.
- Add salary details including basic salary, bonus, and deductions.
- Automatically calculate the net salary.
- Add employee credit (advance payment/loan) records.
- Update the remaining credit balance after repayment.
- View simple reports for employees, salaries, and credits.
